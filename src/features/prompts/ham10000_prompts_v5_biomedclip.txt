# HAM10000 Concept Prompts - VERSION 5 (BiomedCLIP-targeted)
#
# Rationale: BiomedCLIP was pretrained on PubMed Central image-caption pairs
# from biomedical literature. Its PubMedBERT text encoder is tuned for the
# linguistic distribution of medical paper captions, which differs substantially
# from the web image-caption distribution standard CLIP was trained on.
#
# Diagnostic results on BiomedCLIP + v4 prompts (which were optimized for
# ViT-B/32) showed:
#   - Strong gains on non-melanoma classes: bkl (+0.207), akiec (+0.196),
#     vasc (+0.260), df (+0.098), nv (+0.070), bcc (+0.046)
#   - Essentially no gain on melanoma: mel (+0.005)
#
# v5_biomedclip targets the melanoma gap specifically. The mel-targeted and
# nv-anchor concepts use new templates designed to match the PubMed dermatology
# caption distribution:
#   - bt1: "Dermoscopy of [disease] showing [feature]" (PubMed-caption style)
#   - bt2: "Dermoscopic image of [disease] demonstrating [feature]" (alternative
#         caption-style with explicit disease anchor)
#   - bt3: "[Feature description] in dermoscopy of [disease]" (feature-first
#         with disease context)
#
# The disease anchor is "melanoma" for melanoma-targeted concepts and
# "melanocytic nevus" for the benign-nevus-anchor concepts (symmetric_uniform,
# reticular_network).
#
# All other concepts (bkl, akiec, bcc, df, vasc, healthy_skin) retain their v4
# wording — these performed well under BiomedCLIP with v4 prompts and do not
# need rephrasing.
#
# IMPORTANT NOTE ON THE NEVUS ANCHORS:
# An earlier ViT-B/32 iteration (v2) tried "benign melanocytic nevus" framing
# and failed because the BPE tokenizer fragment "melanocytic" activated on
# melanoma images. BiomedCLIP uses PubMedBERT, which encodes "melanocytic
# nevus" and "melanoma" as semantically distinct entities from its literature
# pretraining. This is an empirically testable hypothesis: if v5_biomedclip
# improves nv-anchor concept performance, BiomedCLIP handles the medical
# taxonomy correctly; if not, the same lexical-fragment issue may apply.
#
# Format: each concept has an ID, a short name, a tier, and 3 phrasing variants.
# Template labels (bt1/bt2/bt3 for BiomedCLIP variants, t1/t2/t3 for v4-style
# variants) are normalized to t1/t2/t3 in the prompt loader.
#
# Total: 24 concepts x 3 phrasings = 72 prompts
