# HAM10000 Concept Prompts - VERSION 6 (FINAL)
#
# Rationale: v6 consolidates the prompt set from 3 templates per concept
# (72 prompts) down to a single coherent prompt per concept (24 prompts).
# This is the operative feature set for NAM training.
#
# Iteration history leading to v6:
#   v1: original 3-template prompts, generic dermoscopy phrasing
#       (e.g. "a dermoscopic image of an asymmetric skin lesion")
#   v2: failed "melanocytic nevus" anchor under ViT-B/32 (shared lexical
#       fragment activated on melanoma)
#   v3: visual-descriptor framing for nv anchors (partial success)
#   v4: final ViT-B/32 prompt set (5/8 paired contrasts)
#   v5_biomedclip: PubMed-caption-style phrasing for melanoma + nv-anchor
#       concepts under BiomedCLIP (mel AUC improved by +0.055)
#   v6 (this file): all 24 concepts in a single coherent style
#       "Dermoscopy of [target class] showing [feature]"
#
# Why a single template per concept:
#   - During prompt iteration we maintained 3 templates per concept to
#     enable template-level sensitivity analysis. That analysis is complete
#     and reported in the appendix. For NAM training, the 72-feature matrix
#     is over-parameterised (templates are highly correlated for most
#     concepts; baseline NAM showed overfitting with 72 features).
#   - Inspection of the per-prompt class-mean heatmap (Appendix Figure X)
#     showed that for most concepts the 3 templates carry similar signal,
#     but for a subset (ulceration, milia_like_cysts, red_lacunae,
#     central_white_patch, blue_grey_ovoid_nests), the plain-English t3
#     templates leaked spurious signal onto the vasc class because their
#     generic descriptors ("open wound", "milky spots", "white centre")
#     accidentally matched bright structureless vascular lesions.
#   - Averaging/ensembling templates would smear these signal differences.
#     Picking a single coherent technical template style avoids both the
#     parameter inflation and the template-leakage problem.
#
# Why disease-anchored phrasing:
#   - BiomedCLIP's PubMedBERT text encoder is pretrained on biomedical
#     literature, where dermoscopy is conventionally described in
#     disease-anchored captions ("Dermoscopy of melanoma showing ...").
#   - v5_biomedclip already validated this empirically for melanoma
#     concepts (+0.055 mean AUC on mel vs generic phrasing).
#   - v6 extends the same convention to all 24 concepts for stylistic
#     uniformity.
#
# Format: each concept has one prompt. The "t1" label is preserved for
# backward compatibility with the prompt_loader.py interface; templates t2
# and t3 are intentionally not provided in v6. The prompt loader should be
# updated to handle single-template concepts without error.
