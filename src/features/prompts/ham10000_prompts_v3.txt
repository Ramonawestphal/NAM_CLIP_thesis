# HAM10000 Concept Prompts - VERSION 3
#
# Iteration history:
#   v1: original prompts based on ABCD rule + 7-point checklist + LLM-augmented
#       concepts. 4/8 paired contrasts passed.
#   v2: tried "benign melanocytic nevus" framing for symmetric_uniform and
#       "delicate uniform pigment grid" framing for reticular_network.
#       Result: 2/8 paired contrasts passed — WORSE. The word "melanocytic"
#       activated for melanoma images as well as nevi (both are melanocytic
#       lesions), demonstrating that adding domain-specific terminology can
#       hurt discrimination when the terminology spans both target classes.
#   v3 (this file): reverts to visual-descriptor framing for both concepts.
#       For symmetric_uniform: emphasizes "small + round + smooth edge"
#       (visual properties melanomas typically lack). For reticular_network:
#       describes the visual structure ("fine soft mesh", "thin evenly
#       spaced lines") rather than abstract regularity qualifiers.
#       All other concepts unchanged from v1.
#
# Format: each concept has an ID, a short name, a tier, and 3 phrasing variants.
# Tier 1 = ABCD rule (Stolz et al., 1994)
# Tier 2 = 7-point checklist (Argenziano et al., 1998)
# Tier 3 = LLM-augmented class-discriminative concepts
# Template 1 = "dermoscopic image" (technical, CLIP-style)
# Template 2 = "dermatoscopic image" (alternative technical phrasing)
# Template 3 = "close-up photograph" (plain English, general CLIP)
#
# Total: 24 concepts x 3 phrasings = 72 prompts

# ============================================================
# TIER 1: ABCD RULE OF DERMATOSCOPY
# ============================================================

[asymmetry]
tier: 1
t1: a dermoscopic image of an asymmetric skin lesion
t2: a dermatoscopic image showing a skin lesion with asymmetric shape along both axes
t3: a close-up photograph of a mole that is asymmetric in outline

[border_irregularity]
tier: 1
t1: a dermoscopic image with an irregular and asymmetric border
t2: a dermatoscopic image showing a skin lesion with jagged, notched borders
t3: a close-up photograph of a mole with poorly defined edges

[colour_variation]
tier: 1
t1: a dermoscopic image of a lesion with multiple colours
t2: a dermatoscopic image showing variegated pigmentation including brown, black, and red
t3: a close-up photograph of a skin lesion with uneven color distribution

[diameter_large]
tier: 1
t1: a dermoscopic image of a large skin lesion greater than six millimetres
t2: a dermatoscopic image of a wide pigmented lesion
t3: a close-up photograph of a sizeable mole
