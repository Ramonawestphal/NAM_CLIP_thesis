# Chest X-ray Concept Prompts - VERSION 2 (targeted revision)
#
# Dataset: Kermany et al. (2018) Chest X-ray Pneumonia
#   - 5,856 paediatric chest X-rays
#   - Binary task: pneumonia vs normal (~74% pneumonia)
#   - BiomedCLIP encoder (ViT-B/16 + PubMedBERT)
#
# ──────────────────────────────────────────────────────────────────────
# Iteration history
# ──────────────────────────────────────────────────────────────────────
# v1 (chestxray_prompts_v1_biomedclip.txt): 19-concept set, three tiers.
#   Per-class diagnostic on the train pool found 14/19 concepts MATCH
#   expected target class. Correlation diagnostic found mean |r| = 0.449,
#   with a degenerate normal-anchor block (within-block mean |r| = 0.92)
#   and an interstitial block (within-block mean |r| = 0.88).
#
# v2 (this file): conservative targeted revision.
#   - 5 concepts revised: those that FAILED per-class discrimination in v1
#     (focal_opacity, hyperinflation) and those with noise-level spread
#     (dense_segmental_opacity, bilateral_interstitial_pattern,
#     round_pneumonia).
#   - 1 concept dropped: no_focal_opacity. The v1 diagnostic showed r=0.96
#     with clear_lung_fields. The v1 prompt ("Chest radiograph of normal
#     lungs showing no focal opacity or consolidation") is structurally a
#     negation of pneumonia rather than a description of normal lung
#     anatomy, and contributes no distinct semantic content beyond
#     clear_lung_fields.
#   - 13 concepts unchanged from v1.
#   - Total: 18 concepts.
#
# ──────────────────────────────────────────────────────────────────────
# Design rationale for what was NOT revised
# ──────────────────────────────────────────────────────────────────────
# The normal-anchor block remains highly correlated within itself (4
# remaining concepts after dropping no_focal_opacity; expected mean |r|
# still high). The interstitial block remains internally correlated. This
# is preserved deliberately:
#
#   - The pre-registered hypotheses H1-H5 were formulated on v1's
#     structure. Wholesale revision to break up correlated blocks would
#     have crossed into hypothesis-tuning on observed data.
#   - The thesis claim being tested is whether the *input correlation
#     diagnostic predicts the sparsity selection outcome*. The diagnostic
#     is more informative if measured on prompts that were not
#     engineered to control the correlation structure.
#   - The within-block correlation observed in v1 is itself a finding
#     about how BiomedCLIP encodes related radiology vocabulary; it
#     parallels the HAM10000 atypia block at r > 0.9 and is reported as
#     a result rather than fixed.
#
# Revisions in v2 are limited to concepts that failed to function as
# individual features (wrong-class argmax or noise-level spread). This
# parallels the HAM10000 v7 prompt-ablation logic: revise concepts where
# the empirical evidence of failure is concrete and per-concept.
#
# ──────────────────────────────────────────────────────────────────────
# Revision principles applied to the 5 revised concepts
# ──────────────────────────────────────────────────────────────────────
#   1. Replace radiologist's interpretation words ("infiltrate",
#      "segmental opacity") with concrete visual descriptors of what is
#      seen on the film.
#   2. Disease-anchor every revised concept; the v1 success of the
#      anchored normal block plus the HAM10000 v5_biomedclip → v6 finding
#      both support disease anchoring.
#   3. Where a concept anchored on the wrong class in v1, add language
#      that distinguishes the pathologic finding from the normal-anatomy
#      reading.
#
# Format: each concept has one prompt labelled t1, matching v1 INI
# format. No changes to prompt_loader.py needed.

# v1 prompt: "Chest radiograph of bacterial pneumonia showing a dense segmental opacity"

# Failure mode: near-zero spread indicates the prompt was not meaningfully
# resolved by BiomedCLIP. "Segmental opacity" is anatomic-functional
# jargon with limited representation in image-caption training data.

# (whiteness, spatial confinement) rather than the anatomic-functional
# concept.

[round_pneumonia]
# REVISED
# v1 prompt: "Chest radiograph of bacterial pneumonia showing a rounded well-circumscribed opacity"
# v1 result: MISMATCH (argmax=normal 0.350, bacteria 0.330, virus 0.340;
#            spread=0.021)
# Failure mode: "rounded well-circumscribed opacity" plausibly matched
# normal rounded anatomic shadows (heart, hilum, thymus in paediatric

[hyperinflation]
# REVISED
# v1 prompt: "Chest radiograph of viral pneumonia showing hyperinflation of the lungs"
# v1 result: MISMATCH (argmax=normal 0.439, bacteria 0.397, virus 0.419;
#            spread=0.043 — largest mismatch margin in the diagnostic)
# Failure mode: "hyperinflation of the lungs" anchored on normal
# well-aerated films. The encoder may parse "hyperinflation" as
# "fully-inflated lungs" without the pathologic connotation. The viral
# class has lower mean than normal by 0.02, indicating real
# anti-discrimination.
# Revision strategy: specify visible pathologic signs of hyperinflation
# (over-expanded lungs, flattened diaphragm domes) that do not appear in
# normal-aeration descriptions.
tier: 2
t1: Chest radiograph of viral pneumonia showing overexpanded lungs with flattened diaphragm domes
