# HAM10000 Concept Prompts - VERSION 4
#
# Iteration history:
#   v1: original prompts based on ABCD rule + 7-point checklist + LLM-augmented
#       concepts. 4/8 paired contrasts passed. Mean auc_max = 0.617.
#   v2: tried "benign melanocytic nevus" framing for symmetric_uniform and
#       "delicate uniform pigment grid" framing for reticular_network.
#       Result: 2/8 paired contrasts passed — WORSE. The word "melanocytic"
#       activated for melanoma images (both nevi and melanomas are melanocytic
#       lesions), demonstrating that adding domain-specific terminology can
#       hurt discrimination when the terminology spans both target classes.
#   v3: reverted both to visual-descriptor framing. symmetric_uniform used
#       "small + round + smooth edge" (Candidate F); reticular_network used
#       "fine soft brown mesh" / "thin evenly spaced lines" (Candidate G).
#       Result: 4/8 passed but composition shifted — Candidate F (sym_uniform)
#       improved all four symmetric_uniform-related contrasts by +0.0144;
#       Candidate G (reticular_network) made things slightly worse by
#       activating on melanoma images more than the v1 wording.
#   v4 (this file): keeps Candidate F for symmetric_uniform (the working
#       change) and reverts reticular_network to v1 (the failed change).
#       Expected: 5/8 paired contrasts pass.
#
# Design principle for v4 (relevant for methodology section): prompts are
# optimised for clinical fidelity and directional correctness on paired
# contrasts, NOT for single-prompt discriminative power. Maximising the
# per-prompt one-vs-rest AUC would risk collapsing the diagnostic decision
# into a single dominant feature, defeating the purpose of the additive
# decomposition the NAM is meant to expose. The goal is a feature space in
# which clinically meaningful concepts each carry modest signal, so that the
# NAM has room to learn nonlinear shape functions and concept interactions.
#
# Format: each concept has an ID, a short name, a tier, and 3 phrasing variants.
# Tier 1 = ABCD rule (Stolz et al., 1994)
# Tier 2 = 7-point checklist (Argenziano et al., 1998)
# Tier 3 = LLM-augmented class-discriminative concepts
# Template 1 = "dermoscopic image" (technical, CLIP-style)
# Template 2 = "dermatoscopic image" (alternative technical phrasing)
# Template 3 = "close-up photograph" (plain English, general CLIP)
#
# Total: 24 concepts x 3 phrasings = 72 prompts

tier: 1

t2: a dermatoscopic image showing a skin lesion with asymmetric shape along both axes
t3: a close-up photograph of a mole that is asymmetric in outline
